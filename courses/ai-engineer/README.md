# Track 1: AI Engineer (Generalist)

**Focus:** Architecting and optimizing "Brains" that can see, reason, and adapt.

As an AI Engineer for drone systems, you will master the end-to-end lifecycle of building, training, and optimizing deep learning models — from Vision Transformers that process aerial imagery to production-ready models compressed for edge deployment.

## Learning Outcomes

By completing this track, you will be able to:
- Design and implement Vision Transformer architectures for aerial image analysis
- Fine-tune large language models using LoRA/QLoRA on consumer-grade GPUs
- Build multi-modal systems that fuse image and text for natural language drone queries
- Optimize models from FP32 to FP8/INT4 precision with minimal accuracy loss
- Apply knowledge distillation to create lightweight student models
- Prune model weights to reduce size by 40%+ while maintaining accuracy

## Prerequisites

- PyTorch fundamentals (tensors, autograd, nn.Module)
- Understanding of CNN architectures (ResNet, EfficientNet)
- Basic NLP concepts (tokenization, embeddings)
- Linear algebra (matrix multiplication, eigenvalues)
