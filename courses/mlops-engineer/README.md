# Track 2: ML Platform Engineer (MLOps)

**Focus:** Building the automated "Factory" where models are born and maintained.

As an ML Platform Engineer, you build the infrastructure that enables AI Engineers to train, deploy, and monitor models at scale. You ensure every model is reproducible, every deployment is reliable, and every failure is caught before it impacts drone operations.

## Learning Outcomes

By completing this track, you will be able to:
- Deploy GPU-enabled Kubernetes clusters with auto-scaling
- Script reproducible infrastructure with Terraform/Pulumi
- Build modular ML pipelines with Kubeflow
- Version datasets and models with DVC
- Serve multi-framework models with NVIDIA Triton
- Monitor model drift and system health with Prometheus/Grafana/Evidently
